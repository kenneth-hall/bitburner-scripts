# docs/phases/ — shipped phase docs

Archive of the brainstorm (`.features.md`) and spec (`.spec.md`) docs for **completed** phases.
A phase's docs live in the repo root while it's in flight; when it ships they graduate here.

- **What shipped, when, and where to look** → [`CHANGELOG.md`](CHANGELOG.md) is the index. This
  README deliberately does *not* repeat that list — one source of truth, not two to keep in sync.
- **Naming + the three-stage workflow** → see `CLAUDE.md` › *Development workflow*
  (`phase-NN-slug.<stage>.md`, zero-padded so files sort chronologically).

Active/unshipped phase docs stay in the repo root, not here.
